# 03 地図画面（トップページ `/`）

## 概要

日本地図（SVG）を表示し、都道府県クリックでお土産パネルを開く。
お土産が登録済みの都道府県は色付きで視覚的に区別する。

## タスク

### 日本地図 SVG の組み込み

- [x] 都道府県ごとにパス分割された日本地図 TopoJSON を用意する（`public/japan.topojson` / dataofjapan/land）
- [x] `components/JapanMap.tsx` として React コンポーネント化する（`react-simple-maps` 使用、`dynamic` + `ssr: false`）
- [x] 各都道府県の `<path>` に都道府県名を `aria-label` 属性として付与する

### 色付け（お土産あり都道府県）

- [x] Supabase から locations と items を JOIN して取得する
- [x] お土産ありの都道府県パスに青色（`#3B82F6`）を適用する
- [x] 選択中の都道府県は濃い青（`#1D4ED8`）で強調する
- [x] お土産なしの都道府県はグレー（`#D1D5DB`）で表示する

### 都道府県クリック → パネル表示

- [x] 都道府県クリック時にサイドパネルを表示する（同じ都道府県を再クリックで閉じる）
- [x] パネルに選択した都道府県のお土産一覧（名前・購入日）を表示する
- [x] お土産がない場合は「まだお土産がありません」を表示する
- [x] パネルから `/locations/[id]` へのリンクを設置する（「詳細・編集 →」）
- [x] パネルに閉じるボタン（×）を設置する

### カスタム場所カードリスト

- [x] `is_custom = true` の locations を地図下にカードリストとして表示する
- [x] カードをクリックすると `/locations/[id]` へ遷移する
- [x] カスタム場所が 0 件の場合はカードリストを非表示にする
- [x] カードにお土産件数バッジを表示する

### レスポンシブ対応

- [x] スマートフォン表示で地図が幅いっぱいに表示される（`width: 100%`）
- [x] PC 表示でパネルが地図の右側に表示される（`md:flex-row`）
- [x] スマートフォン表示でパネルが地図の下に表示される（`flex-col`）

## 完了条件

- [x] 地図が表示され、都道府県をクリックするとパネルが開く
- [x] お土産が登録済みの都道府県が色付きで表示される（DB からデータ取得）
- [x] カスタム場所がカードリストで表示される（件数バッジ付き）
