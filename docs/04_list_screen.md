# 04 一覧画面（`/list`）

## 概要

全お土産をリスト表示する画面。場所での絞り込みと場所名順ソートに対応する。

## タスク

### ページ作成

- [x] `app/list/page.tsx` を作成する（Server Component）
- [x] Supabase から全 items を locations と JOIN して取得する（`Promise.all` で locations も並列取得）

### リスト表示

- [x] お土産名・場所名・購入日・メモをリスト形式で表示する
- [x] 件数が 0 の場合は「お土産がまだ登録されていません」を表示する

### 絞り込み（場所）

- [x] 場所（都道府県またはカスタム場所名）で絞り込むセレクトボックスを設置する
- [x] 「すべての場所」を選択すると絞り込みを解除する
- [x] 絞り込み状態は URL クエリパラメータ (`?location=`) で管理する

### ソート（場所名順）

- [x] 場所名（`locations.name`）の昇順でデフォルトソートする（`localeCompare` 日本語対応）
- [x] ソート切り替えボタンを設置する（昇順 ↑ / 降順 ↓）
- [x] ソート状態は URL クエリパラメータ (`?sort=desc`) で管理する

### 各お土産のリンク

- [x] 各行・カードから `/locations/[id]` へ遷移できる「詳細 →」リンクを設置する

### レスポンシブ対応

- [x] PC（`md:` 以上）でテーブル形式で表示する
- [x] スマートフォン（`md:` 未満）でカード形式で表示する

## 完了条件

- [x] 全お土産が一覧表示される
- [x] 場所で絞り込みができる（URL パラメータ連動）
- [x] 場所名順でソートできる（昇順・降順切り替え）
