# 要件定義書 — お土産管理 Web サイト

## 1. プロジェクト概要

日本各地で購入したお土産を個人で管理するための Web アプリケーション。
購入した場所・品名・日付・メモを記録し、日本地図や一覧画面から確認できる。

---

## 2. 機能要件

### 2-1. 認証

- 固定パスワード 1 つで全員同じアクセス権限を付与する
- パスワードは環境変数 `AUTH_PASSWORD` で管理する
- 未認証時はすべてのページを `/login` へリダイレクトする
- 認証状態は Cookie で保持する

### 2-2. 場所管理

- 初期データとして日本の 47 都道府県を登録済みの状態で起動する
- 場所の粒度は「都道府県 + 任意の市区町村」とする
- 都道府県・市区町村に属さないカスタム場所（例: 富士山）を任意で追加できる
- 場所ごとにお土産をまとめて管理する

### 2-3. お土産管理

各お土産に以下の情報を記録する。

| 項目 | 必須 | 備考 |
|------|------|------|
| お土産名 | ○ | |
| 購入場所 | ○ | locations テーブルの ID を参照 |
| 購入日 | ○ | |
| メモ | — | 任意の自由記述 |

- お土産の追加・編集・削除ができる

### 2-4. 地図画面（トップページ `/`）

- 日本地図（SVG）を背景に表示する
- 都道府県をクリックすると、その都道府県で購入したお土産一覧をパネルで表示する
- お土産が 1 件以上登録されている都道府県は、地図上で視覚的に区別する（色付け等）
- カスタム場所（都道府県に属さない場所）は地図下にカードリストとして表示する（MVP ではマップピン表示は省略）

### 2-5. 一覧画面（`/list`）

- 全お土産をリスト表示する
- 場所（都道府県またはカスタム場所）での絞り込みができる
- 場所名順でのソートができる

### 2-6. 場所詳細画面（`/locations/[id]`）

- 特定の場所に紐づくお土産一覧を表示する
- お土産の追加・編集・削除ができる

---

## 3. 非機能要件

- 個人運営を前提とする（マルチユーザー管理は不要）
- レスポンシブデザイン対応（PC・スマートフォン）
- 写真アップロード機能は MVP では対象外
- 価格記録機能は MVP では対象外

---

## 4. データモデル

### locations（場所）

| カラム | 型 | 説明 |
|--------|----|------|
| id | uuid (PK) | |
| prefecture | text \| null | 都道府県名。カスタム場所は null |
| city | text \| null | 市区町村名（任意） |
| name | text | 表示名（例: "北海道 函館市"、"富士山"） |
| is_custom | boolean | カスタム場所フラグ |
| created_at | timestamptz | |

### items（お土産）

| カラム | 型 | 説明 |
|--------|----|------|
| id | uuid (PK) | |
| location_id | uuid (FK) | locations.id を参照 |
| name | text | お土産名 |
| purchased_at | date | 購入日 |
| memo | text \| null | メモ（任意） |
| created_at | timestamptz | |

---

## 5. 画面構成

| パス | 画面名 | 概要 |
|------|--------|------|
| `/login` | ログイン | パスワード入力・Cookie 発行 |
| `/` | 地図画面 | 日本地図 + カスタム場所カードリスト |
| `/list` | 一覧画面 | 全お土産リスト・絞り込み・ソート |
| `/locations/[id]` | 場所詳細 | 特定場所のお土産一覧・CRUD |

---

## 6. 技術スタック

| 区分 | 採用技術 |
|------|----------|
| フレームワーク | Next.js 15 (App Router) |
| 言語 | TypeScript |
| スタイリング | Tailwind CSS |
| データベース | Supabase (PostgreSQL) |
| 認証 | Next.js Middleware + Cookie（固定パスワード） |
| 地図 | 日本地図 SVG（都道府県単位でパス分割） |
| デプロイ | Vercel |
